## Prerequisites
 - Node.js
 - npm
 - source code
   
## Installation guide
Download and unpack the source code. <br>
Navigate to the directory where you've unpacked the source code.
Open the terminal
### <pre> cd backend </pre>
### <pre> npm install </pre>
### <pre> npm start </pre>
Open another termial 
### <pre> cd frontend </pre>
### <pre> npm install </pre>
### <pre> npm run dev </pre>
You are ready to go
